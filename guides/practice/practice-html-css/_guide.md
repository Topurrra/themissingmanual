# HTML & CSS Practice

Twelve short, hands-on HTML and CSS lessons: start with a heading and a
paragraph, learn what a class is for, then build the layouts you actually build
at work - a row of cards, a centred box, a nav bar, a gallery. Every lesson
gives you a task, a live preview, and a Run button. Your page redraws as you
run it, and the checks tell you what is still off. Everything runs in your
browser.

The last four are the ones that make people quietly hate CSS. Nothing errors -
CSS has no traceback, so a rule that never applies looks exactly like a rule
you forgot to write. A box that overflows because padding was added to a width,
a colour that loses to a more specific selector, a margin that escapes its
parent, and a card that a single long word tears open.

Start with lesson 1. You can leave and come back any time - your code is saved locally.
