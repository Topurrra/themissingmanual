# Git Practice

Five short, hands-on Git lessons: make your first commit, build a real commit
history, branch and switch, merge your work back in, and learn to read
`git status` like a pro. Each lesson gives you a task and a tiny in-browser
terminal - not a code editor. You type one real `git` command per line, run
it, and get checked instantly against a real repository running entirely in
your browser.

Start with lesson 1. You can leave and come back any time - your code is saved locally.
