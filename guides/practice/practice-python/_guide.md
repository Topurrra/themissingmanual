# Python Practice

Sixteen short, hands-on Python lessons: start with your first print statement,
build up through error handling, generators, classes, and recursion, debug a
real traceback, then finish with a function-writing capstone. Each lesson
gives you a task and a Run button - your code is checked against real tests
the moment you run it. Everything runs in your browser.

Start with lesson 1. You can leave and come back any time - your code is saved locally.
